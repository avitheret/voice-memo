Opens this app from Siri. Say "Hey Siri, Record" — recording starts automatically. Works with AirPods or any Bluetooth mic.

1
Host this file on GitHub Pages, Netlify, or any web server — note the URL.
2
On iPhone, open the Shortcuts app → tap + to create a new shortcut.
3
Tap the shortcut name at the top → rename it Record.
4
Tap Add Action → search Open URLs → select it.
5
Set the URL to https://avitheret.github.io/voice-memo/?autostart=1
6
Tap Done. Now say "Hey Siri, Record" — the app opens and starts recording immediately.

On desktop (Chrome/Edge), use the wake word button above instead — no Siri needed.

No web hosting needed. Creates a standalone Siri Shortcut that records, transcribes, and saves directly to iCloud Drive as a .txt file.

1
Open Shortcuts → tap + → name it Record.
2
Add action: Dictate Text — set Language to English, stops on: Silence.
3
Add action: Format Date — choose Custom, format: MMM d yyyy, h:mm a.
4
Add action: Text — enter: [Dictated Text]↵— [Formatted Date]↵↵ (use variable chips).
5
Add action: Get File → iCloud Drive → Voice Memos.txt. Enable Error If Not Found: OFF.
6
Add action: Combine Text → combine [File Contents] + [Text from step 4], separator: New Lines.
7
Add action: Save File → iCloud Drive → filename Voice Memos.txt, Overwrite: ON.
8
Tap Done. Say "Hey Siri, Record" — iPhone records, transcribes, and appends to your file.

Your memos accumulate in iCloud Drive → Voice Memos.txt, visible in the Files app on all devices.
